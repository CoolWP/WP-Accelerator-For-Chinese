WordPress Accelerator for Chinese

== Description ==

WordPress Accelerator for Chinese.

*   自定义 jQuery 以及 jQuery Migrate 的URL;
*   自定义是否清理头部输出;
*   前后台字体优化;
*   可选地移除 WordPress 自身以及其默认主题(2014-2016)加载的谷歌字体;
*   自定义是否启用 emoji ;
*   如果启用 emoji, 可自定义是采用 WordPress默认的BASE URI (由于某种神秘原因,s.w.org 通常会是404或者很慢),还是采用自定义的 BASE URI.
*   防止自ping;
*   阻止 pingback.ping .
== Installation ==


就像安装其它插件那样安装并启用，在插件列表该插件下有"设置"按钮，到设置页面进行下设置，就行了。

Install and activated it as normal, you will find a "Setting" button under this plugin on the plugins screen.
