WordPress Accelerator for Chinese

== Description ==

WordPress Accelerator for Chinese.

自定义静态加速，弥补七牛 10GB 限制的不足，以及其它一些性能优化,详见[项目主页](http://coolwp.com/wp-accelerator-for-chinese.html).

== Installation ==


就像安装其它插件那样安装并启用，在插件列表该插件下有"设置"按钮，到设置页面进行下设置，就行了。

Install and activated it as normal, you will find a "Setting" button under this plugin on the plugins screen.
